    SensDash Application using XMPP data streaming and AngularJS

    What is it all about:

    SensDash is a dashboard with a list of connected Hardware (temperature, humidity, pressure etc.) and Software (feeds, tweets, rss) sensors. All code was done following the MVC pattern, for that AngularJS Framework was used.

    The goals are:

    1. Connect any type of sensor through designed universal approach and retrieve the streaming data through the XMPP MUC/PubSub.
    2. Dynamically and in runtime add new sensor and create a visual container/template for a new sensor.
    3. Implement subscription to a service/sensor and online retrieving of its data.
